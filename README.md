# BioVita Dermo — Site protótipo

Site estático em HTML, CSS e JavaScript puro para apresentação escolar da BioVita Dermo.

## Arquivos

- `index.html` — estrutura do site
- `style.css` — identidade visual e responsividade
- `script.js` — menu, pesquisa, diagnóstico, plano e ficha de interesse
- `assets/` — logos e mockups SVG originais dos produtos

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie todos os arquivos deste projeto para a raiz do repositório.
3. Vá em **Settings > Pages**.
4. Em **Build and deployment**, escolha **Deploy from a branch**.
5. Selecione branch `main` e pasta `/root`.
6. Clique em **Save**.

Depois de alguns minutos, o GitHub vai gerar o link público do site.

## Observação

O site usa apenas recursos locais. Não depende de servidor, banco de dados, API externa ou bibliotecas.
